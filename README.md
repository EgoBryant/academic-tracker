# Academic Tracker

Веб-приложение для учета успеваемости студентов. Проект позволяет управлять учебными предметами, оценками и помогает отслеживать академическую успеваемость в одном месте.

---

# О проекте

Academic Tracker — это веб-приложение, разработанное в рамках командной стажировки. Основная цель проекта — предоставить студентам удобный инструмент для хранения информации о предметах, оценках и учебном процессе.

Проект разработан с использованием современного клиент-серверного подхода с разделением на frontend и backend.

---

# Основные возможности

- Регистрация пользователей
- Авторизация пользователей
- Управление учебными предметами
- Управление оценками
- Автоматический расчет среднего балла
- Просмотр информации по каждому предмету
- Валидация пользовательского ввода
- Обработка ошибок
- Современный пользовательский интерфейс

# Архитектура проекта

Проект разделен на две независимые части:

- **Frontend** — пользовательский интерфейс.
- **Backend** — REST API и работа с базой данных.

Такой подход позволяет независимо развивать клиентскую и серверную части приложения.

# База данных
